# Health Indicators: Next

> The most pointless mod port you will ever install.

Once upon a time, the original [Health Indicators](https://modrinth.com/mod/health-indicators) had been dead for **6 whole months**. No updates. Silence. A tombstone.

So I did the noble thing: I hand-migrated the entire mod to Minecraft **26.2** — fighting through Yarn-to-Mojang mappings, the great `hurt()` -> `hurtServer()` schism, a Forgix plugin that couldn't read Java 25, and a `fabric.mod.json` that insisted on a mod named `fabric` that no longer exists.

It worked. Everything compiled. Everything ran. Victory.

**Then, exactly one day later, the original mod rose from its grave and published its own 26.2 build.**

So here we are. There are now two of them. This is the one nobody asked for, maintained by the one developer who learned the hard way that the universe has a sense of humor.

## Should you install this?

No. Install the [original](https://modrinth.com/mod/health-indicators). It's probably better maintained now.

But if you appreciate a good monument to wasted effort — welcome aboard. It shows floating health bars (or numbers) above Players and Mobs, exactly like the real one, plus 100% bonus emotional damage (to the developer, not the mobs).
